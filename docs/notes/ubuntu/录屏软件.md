### 1. Kazam

快捷键

start recording: Super + Control + r
pause recording: Super + Control + p
finish recording: Super + Control + f
show Kazam: Super + Control + s
quit Kazam: Super + Control + q
Note: "Super" is usually this "Windows logo" key.
