Ubuntu 系统有很多键盘快捷键可用，其中就包括：包括 GNOME 快捷键（例如 Windows 键打开 App）、Nautilus 快捷方式（例如 Ctrl + T 打开一个新选项卡）和 Ubuntu Dock 快捷方式（Super + 1-9）等等。
